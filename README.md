User Management Site 👨‍💻

A full-stack user management application built to practice CRUD operations, authentication, and modern web development using the MERN stack.

This project allows users to manage user data efficiently with features like adding, updating, deleting, and viewing users through a clean and responsive interface.

Repository: User-Management-Site

🚀 About The Project

The main goal of this project was to understand how frontend and backend applications work together in a real-world full-stack application.

The application focuses on:

Managing user information
Performing CRUD operations
Building REST APIs
Connecting React frontend with Express backend
Working with MongoDB database
Creating a responsive user interface
✨ Features
➕ Add New Users
📝 Update User Details
❌ Delete Users
👀 View All Users
🔍 Search & Manage Data
📱 Responsive UI
⚡ Fast and Smooth Frontend
🌐 REST API Integration

⚙️ Getting Started
1. Clone the Repository
git clone https://github.com/shivakumar109/User-Management-Site.git
cd User-Management-Site
🔧 Backend Setup

Move into the backend folder:

cd server

Install dependencies:

npm install

Create a .env file inside the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string

Run the backend server:

npm run dev
💻 Frontend Setup

Move into the frontend folder:

cd client

Install dependencies:

npm install

Run the frontend:

npm run dev
